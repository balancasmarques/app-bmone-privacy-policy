Privacy Policy for BM ONE

Introduction

This Privacy Policy explains how the BM ONE application uses the permissions granted by the user and how we protect their privacy.

Information Collected

Our application requires the following permissions:

  • ACCESS_WIFI_STATE: Used exclusively to check the Wi-Fi connection status and connect to the BM ONE machine.
  • ACCESS_NETWORK_STATE: Necessary to determine whether the device is connected to the access point of the BM ONE machine for communication.
  • INTERNET: This permission is not used for internet access. The user must connect to the access point of the BM ONE machine to communicate.

How We Use the Information

The permissions are strictly used to:

  • Establish communication with the BM ONE machine by connecting to its access point.
  • Ensure the proper functioning of the application after the device restarts and re-establish connection with the BM ONE machine.
  • Check the status of the Wi-Fi and network connection to ensure the device is connected to the BM ONE access point.

Data Sharing

The application does not share any personal data or connection information with third parties.
